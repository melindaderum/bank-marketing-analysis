# Bank Marketing Subscription Prediction

Analyzed a dataset of 45,000+ bank customers to predict which customers 
are most likely to subscribe to a term deposit.

## What This Project Does
- Cleans and prepares raw marketing data for analysis
- Explores patterns across customer demographics (job, age, account balance)
- Builds a logistic regression model to predict subscription likelihood
- Identifies high-value customer segments for targeted outreach

## Key Findings
- Retired customers and students had the highest subscription rates
- Customers over 60 were the strongest single predictor
- Higher account balances correlated with higher subscription rates
- Model precision: 67% for identifying likely subscribers

## Tools Used
Python, pandas, numpy, seaborn, matplotlib, scikit-learn

## Data Source
UCI Machine Learning Repository — Bank Marketing Dataset  
https://archive.ics.uci.edu/dataset/222/bank+marketing
